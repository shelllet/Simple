# HEAD 请求
使用 HTTP HEAD 方法从服务器请求 HTTP 标头，就像使用 HTTP GET 方法请求文档一样。HTTP HEAD 和 GET 请求之间的唯一区别是，对于 HTTP HEAD，服务器仅返回标头而不返回正文。
