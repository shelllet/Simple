# 监听 
此动作用来创建TCP或者UDP服务端。

![action](./images/01.png ':size=90%')

## 子动作
> 不支持

## 运行参数


* Host Address： 监听的IP, 可忽略，默认`127.0.0.1`
* Port Number： 监听的端口
* Type： 类型，分为`TCP` 或者`UDP`

## 输出 无


### 脚本调用

```python
import simple;

```
