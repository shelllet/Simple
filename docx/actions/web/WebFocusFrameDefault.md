# 重置框架 
将焦点切换到默认框架。通常操作完 *iframe* 后，可以使用该动作切换回默认的 *html* 页面。

![WebFocusFrameDefault](./images/17.png ':size=90%')

## 运行参数

* 对象
> 浏览器对象
